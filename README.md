## 0x02 编辑环境配置

推荐sublime text（插件简直太爽）。有这么几个markdown的插件：

* `Markdown Preview`，可以在浏览器中预览当前编辑的markdown文件。

* `Markdown Editing`，编辑markdown的皮肤？（应该是吧，可以补全括号引号什么的）

* `Markdown Extended`，支持markdown语法高亮。
